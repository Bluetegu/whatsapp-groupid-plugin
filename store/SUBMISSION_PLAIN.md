=== DETAILED DESCRIPTION ===

Automatically adds WhatsApp group ID information to group info panels on WhatsApp Web. Perfect for Openclaw users who need to configure WhatsApp group IDs quickly and accurately.

Key Features:
• One-click copy: Copy group IDs directly to clipboard with a single click
• Seamless integration: Matches WhatsApp Web's native design perfectly
• Universal support: Works with all group ID formats including dashed IDs (972543343341-1427116328@g.us)
• Privacy-focused: No data collection, tracking, or external servers
• Lightweight: Minimal impact on WhatsApp Web performance
• Secure: Only runs on WhatsApp Web, restricted to web.whatsapp.com

How to Use:
1. Open WhatsApp Web (web.whatsapp.com)
2. Click on any group conversation
3. Click the group name to open group info
4. Find the new "Group ID" section above "Media, links and docs"
5. Click the copy button to copy the group ID

Perfect for:
- Openclaw users configuring WhatsApp group automation
- Developers working with WhatsApp Business API
- System administrators managing group integrations
- Anyone needing quick access to WhatsApp group identifiers

The extension extracts group IDs in clean format needed for external integrations: 123456789@g.us or 123-456-789@g.us

=== SINGLE PURPOSE DESCRIPTION ===

This extension has a single, clear purpose: to extract and display WhatsApp group IDs on WhatsApp Web for easy copying. It adds a "Group ID" section with a copy button to group info panels, allowing users to quickly copy group identifiers for use in external applications like Openclaw. The extension does not modify any other WhatsApp functionality and operates solely to provide access to publicly visible group information.

=== PERMISSIONS JUSTIFICATION ===

activeTab Permission:
Required to access and modify the WhatsApp Web interface to extract group IDs and add the copy functionality. The extension only works on the currently active WhatsApp Web tab.

Host Permission (web.whatsapp.com):
Ensures the extension only operates on WhatsApp Web for security and prevents it from accessing any other websites. This restricts the extension to its intended purpose.
